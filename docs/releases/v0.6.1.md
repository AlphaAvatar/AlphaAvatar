## AlphaAvatar v0.6.1

This release adds visual identity support to AlphaAvatar’s Persona system and improves realtime multimodal identity resolution.

### Highlights

- Added **visual identity support** for the Persona plugin.
- Integrated realtime **face detection and face vector extraction** from camera input.
- Added **face vector matching, retrieval, and persistence** through the Persona VDB layer.
- Added **speaker-face identity fusion**, allowing voice and face signals to resolve and enrich the same user profile.
- Improved Persona cache behavior when resolving temporary users into persistent user identities.
- Added face-based profile attribute updates such as estimated age range and gender.
- Reduced high-frequency realtime face update logs.
- Fixed several bugs related to Persona runtime state, identity merging, VDB loading/saving, and multimodal identity updates.
- Improved the **Persona** and **Memory** plugin README files for clearer setup, configuration, and usage guidance.

### Notes

This version is an important step toward a more persistent and multimodal personal AI assistant. AlphaAvatar can now use both voice and visual signals to better understand who it is interacting with during realtime sessions.
